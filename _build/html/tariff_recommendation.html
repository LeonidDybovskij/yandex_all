Tariff recommendation#

A mobile operator needs a model that analyzes user behavior and offers one of two tariffs: “Smart” or “Ultra”.
It needs a model that solves the classification problem with accuracy >= 0.75.

Conclusions#

  1. Random forest performed the best. Despite the large number of hyperparameters tried, the best model was trained on the average depth.

  2. Decision trees showed the same trend, but their accuracy was lower.

  3. Logistic regression was the worst predictor: this model failed to pass the required threshold on the validation sample.

  4. Blind guessing, not based on parameters, gives 72% correct answers. All models show higher accuracy.

  5. With the best hyperparameters (depth 10, number of trees – 18) random forest showed accuracy 0.82 on the test sample. The model looks the most promising among the considered ones (maybe it will become even better with increasing depth).
